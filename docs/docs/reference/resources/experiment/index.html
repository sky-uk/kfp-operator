Experiment

The Experiment resource represents the lifecycle of Experiments on Kubeflow Pipelines, and can be configured as follows:

apiVersion: pipelines.kubeflow.org/v1alpha1
kind: Experiment
metadata:
    name: penguin-experiment
spec:
    description: 'An experiment for the penguin pipeline'

Fields

NameDescription
spec.descriptionThe description of the experiment
